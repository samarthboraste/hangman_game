# 🎮 Hangman Game (Python)

A simple command-line Hangman game built using Python.  
The player has to guess the hidden word one letter at a time before running out of chances.

---

## 📌 Features
- Random word selection from `words.txt`
- User input validation
- Tracks guessed letters
- Limited number of chances
- Displays progress after each guess

---

## 🛠️ Technologies Used
- Python 3
- File handling
- Random module
